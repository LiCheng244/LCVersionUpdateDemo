# VersionUpdateDemo
关于 iOS 版本更新 的小demo

版本更新的工具类

使用方法:

/**
 *  版本更新
 *
 *  @param appid              该app的id (在itunes connect中获取)
 *  @param isShowReleaseNotes 是否显示版本注释
 *  @param controller         要显示的controller
 */

```
 [UpdateVersionTool updateVersionForAppID:@"1111" isShowReleaseNotes:YES showController:self];
```
